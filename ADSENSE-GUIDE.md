## 💰 如何开始真实赚钱

### 第1步：申请 Google AdSense

1. **访问 AdSense**：
   ```
   https://www.google.com/adsense
   ```

2. **注册账号**：
   - 使用你的Google账号登录
   - 填写网站信息
   - 网站URL：`https://techdaily-auto.vercel.app`（或你的Vercel地址）

3. **提交审核**：
   - 填写个人信息（姓名、地址、税务信息）
   - 同意条款
   - 提交审核

4. **等待审核**：
   - 通常需要 **1-2周**
   - Google会检查网站内容、流量、质量
   - 审核期间可以继续优化网站

### 第2步：获取你的发布商ID

审核通过后：

1. 登录 AdSense 后台
2. 在左上角找到你的发布商ID
3. 格式类似：`ca-pub-1234567890123456`
4. 复制这个ID

### 第3步：替换测试ID

**方法A：告诉我你的ID，我帮你替换**

把你的发布商ID发给我，格式：
```
ca-pub-你的数字
```

我会自动替换代码并推送更新。

**方法B：手动替换**

编辑 `pages/index.js`，找到两处：

```javascript
// 第1处：Script标签（大约在第42行）
<Script

  crossOrigin="anonymous"
  strategy="afterInteractive"
/>

// 第2处和第3处：广告位（大约在第62行和第118行）
<ins className="adsbygoogle"
     data-ad-client="ca-pub-3940256099942544"
                            ^^^^^^^^^^^^^^^^
                         替换这个测试ID
     ...
</ins>
```

将所有 `ca-pub-3940256099942544` 替换为你的ID。

## ❓ 常见问题

### Q: 测试广告会显示真实广告吗？

A: 会！Google会显示真实的广告样式，但不计入收益。

### Q: 测试期间点击广告会怎样？

A: 不会有问题，测试ID专门用于测试，不会违反政策。

### Q: 什么时候应该替换成真实ID？

A: 当你的AdSense账号审核通过后，立即替换。

### Q: 可以同时使用多个广告平台吗？

A: 可以！可以同时使用：
- Google AdSense（主要）
- 百度联盟（中文流量）
- Carbon Ads（科技类）

### Q: 广告不显示怎么办？

可能原因：
1. 浏览器安装了AdBlock - 暂时关闭测试
2. 广告还在加载 - 等待几秒
3. 网络问题 - 刷新页面
4. 代码错误 - 查看浏览器控制台

### Q: 如何查看广告收益？

替换真实ID后：
1. 登录 AdSense 后台
2. 查看 "报告" 页面
3. 可以看到展示、点击、收益等数据
